Introdução

O primeiro passo é utilizar um arquivo de contagem bruta de leituras para cada gene, em cada uma das réplicas das diferentes condições utilizadas no experimento que deu origem aos dados do sequenciamento de RNA.

A tabela de contagem é o input para o pacote, sendo obtida através das leituras de RNA-Seq que se alinharam aos genes a partir de uma referência já anotada. Cada célula representa as contagens para um gene \(i\) na amostra \(j\). O DESeq2 utiliza um modelo estatístico em que só são necessárias as contagens brutas.

Nota-se a importância de que cada coluna represente as réplicas biológicas. Se houver réplicas técnicas, recomenda-se somá-las em uma única coluna.

O Bioconductor é uma fonte aberta baseada em linguagem R, iniciada no ano de 2001, cujo objetivo principal é fornecer ferramentas para análise e compreensão de dados de plataformas de alto rendimento na forma de pacotes para o R.

Para utilizar o pacote DESeq2 deve-se criar metadados, que são arquivos de dados sobre outros dados. Um deles trata-se da informação sobre as amostras, contida nas colunas, dessa forma cria-se um arquivo para especificar o intuito do experimento.

Dessa forma, o software verificará, posteriormente, quais foram os tratamentos utilizados no experimento. Para isso, é necessário criar um vetor que relacione as réplicas da matriz de contagem com as condições.


expDesign <- data.frame(row.names = colnames(counts), condition = c(rep("mock",3), rep("hrcc",3)))

Dessa forma \(expDesign\) é um data frame de informação sobre as amostras, onde as linhas ligam-se diretamente com as colunas da tabela de contagem.

Com a matriz de contagem e a informação das amostras, constrói-se o \(DESeqDataSet\):


dds <- DESeqDataSetFromMatrix(countData = counts, colData = expDesign, design = ~condition)

A análise de expressão diferencial no DESeq2 utiliza:

\(K_{ij} \sim NB(\mu_{ij}, \alpha_{i})\)

\(\mu_{ij} = s_{j}q_{ij}\)

As contagens \(K_{ij}\) para o gene \(i\) na amostra \(j\) são modeladas usando uma distribuição binomial negativa com média ajustada \(\mu_{ij}\) e um padrão de dispersão gene específico \(\alpha_{i}\).

A média ajustada é composta por um fator de tamanho específico de amostra \(s_{j}\) e um parâmetro \(q_{ij}\) proporcional à concentração verdadeira de fragmentos para a amostra \(j\).

Emprega-se a função DESeq:

dds  <- DESeq(dds)

Compreendendo o Log2 Fold Change

#Considere duas contagens:

condA <- 30

condB <- 300

#O Fold Change representa a razão entre elas.

foldChange <- condB/condA


Qual o valor do Fold Change?

Ou seja, se a contagem da condição B for maior do que a de A, o Fold Change será maior do que um. Se a contagem em A for maior do que em B, o Fold Change será inferior a um.

#Já que razão também pode ser:

foldChange <- condA/condB


Qual o valor desse Fold Change?

No que implicaria utilizar apenas o Fold Change?

#Vamos utilizar mais contagens???

condA <- seq(from = 30, to = 300, by= 30)

condB <- seq(from = 300, to = 30, by= -30)


foldChange <- condB/condA


dados <- data.frame(condA, condB, foldChange, condA.maior.condB = condA > condB)

Observe os dados.


library("ggplot2")

ggplot(dados, aes(condA, foldChange, colour = condA.maior.condB), size = 8) + geom_point()


Agora podemos utilizar o log2:

log2foldChange <- log2(foldChange)


dados <- data.frame(condA, condB, foldChange, log2foldChange, condA.maior.condB = condA > condB)

#Observe os dados agora

ggplot(dados, aes(condA, log2foldChange, colour = condA.maior.condB), size = 8) + geom_point()

No DESeq2 a função \(plotMA\) mostra o \(log2\) fold change sobre a média das contagens normalizadas. Se o valor do valor-\(p\) ajustado for inferior a 0.1, será colorido de vermelho.

plotMA(res, main ="DESeq2", ylim = c(-2, 2))

identify(res$baseMean, res$log2FoldChange)
